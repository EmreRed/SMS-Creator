### INSTALLATION
```
npm i sms-creator
```
<br />  

### HOW TO USE
```
var smscreator = require('sms-creator');

smscreator.wrap(message_content, message_encode);
```
`message_content` : Entire message content  
`message_encode` : ASCII / LATIN / TURKISH / UNICODE
